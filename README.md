# PPlab2prob3
Să se implementeze o aplicație Polyglot într-un limbaj la alegere (suportat de GraalVM), care să conțină
trei funcții:
 o funcție python care să genereze aleatoriu o listă de 20 de numere întregi;
 o funcție javascript care va afișa lista de numere întregi;
 o funcție R care să sorteze lista respectivă, să elimine primele și ultimele 20% dintre
elemente și să calculeze media aritmetică
